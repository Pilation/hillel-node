# Books Collection CLI

## Commands

### app.js

```bash
node app.js
```

### info.js

```bash
# Default - list all books
node info.js

# Authors
node info.js --author

# Genres
node info.js --genre

# Rating filter
node info.js --rating=4.5
node info.js --rating=4.3
node info.js --rating=4.0

# Add tags
node info.js --tags=classical
node info.js --tags=literature
node info.js --tags=fiction
```
